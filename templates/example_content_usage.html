Setting Up Content in Admin Panel

  1. Go to the admin panel at /admin/content_manager/pagesection/
  2. Create a new Page Section with a unique identifier (e.g., 'home-welcome-text')
  3. Fill in the appropriate fields (title, subtitle, content, image, etc.)
  4. For site-wide settings, go to /admin/content_manager/sitesetting/
  5. Create settings with unique keys like 'contact_email', 'contact_phone', etc.
  6. The content will be automatically available in your templates using the tags shown above
